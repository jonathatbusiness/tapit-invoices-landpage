# TapIT Invoices

**TapIT Invoices** is a lightweight web application designed to generate clean, professional invoices quickly and efficiently.

The project focuses on clarity, usability, and visual consistency, offering customizable themes and a streamlined workflow suitable for freelancers, contractors, and small businesses.

## Key Features

- Clean and minimal invoice layout
- Multiple light and dark visual themes
- Real-time preview
- Export to PDF
- Client-ready, professional formatting
- Accessible typography and layout

## Design Principles

TapIT Invoices is built with a strong emphasis on:

- Readability and visual hierarchy
- Accessibility-friendly font sizing and spacing
- Consistent UI patterns
- Responsive behavior across screen sizes
- Minimal cognitive load for end users

## Technology Stack

- React
- Vite
- Modern CSS (custom properties and responsive typography)

## Project Status

This project is under active development and iteration.  
Features, visuals, and structure may evolve over time as usability and functionality are refined.
